Features

Built for scan-to-BIM workflows

Practical tools for modelers working with laser scans and as-built verification.

Point cloud visibility

Quickly hide or show all point clouds in the active view — no digging through visibility settings.

Pick coordinates on cloud

Click a point on the scan and get precise X, Y, Z coordinates using Revit's native snaps.

Measure on building elements

Pick a point on floors, walls, roofs, stairs, columns, toposolids, and more — in 2D or 3D views.

Export & import markers

Exchange pinpoint markers as CSV or JSON for coordination, QA/QC, and field-to-model workflows.

Installation

Up and running in under a minute

Close Revit before you begin.

Download the zip for your Revit version and extract it anywhere.

Right-click install.ps1 and choose Run with PowerShell, or run it from a PowerShell prompt in that folder:

powershell 
.\install.ps1

Open Revit — a new Scan2BIM ribbon tab will appear.

Windows download note: if the zip is blocked, right-click it, choose Properties, and select Unblock (or run Unblock-File in PowerShell) before extracting.
Manual install: copy Scan2BIM.dll and Scan2BIM.addin to %ProgramData%\Autodesk\Revit\Addins\20XX\ (replace 20XX with your version).
